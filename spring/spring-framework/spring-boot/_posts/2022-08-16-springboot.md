## SpringBoot (Spring과 차이)
- Embed Tomcat 사용 : 스프링부트 내부에 `내장탐캣` 포함
- starter를 통한 `dependency 자동화`
- application.yml(application.properties) 사용
- jar file로 배포 가능

### application.yml
`web.mxl +  root-context.xml + servlet-context.xml`     

- 참고) 설정파일
  - xml : `<name>value</name>`
  - JSON : `{"name":"value"}`
  - yml : `name: value` <- 가벼움
